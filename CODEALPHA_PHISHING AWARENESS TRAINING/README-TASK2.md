# CodeAlpha - Phishing Awareness Training

## Project Overview
This project is a simple Phishing Awareness Training module developed in Python. It helps users identify phishing emails and understand cybersecurity best practices through an interactive quiz.

## Features
- Interactive phishing awareness quiz
- Identifies suspicious emails and websites
- Educates users about phishing attacks
- Displays final score and feedback

## Technologies Used
- Python 3

## How to Run

```bash
python phishing_training.py
```

## Sample Output

```
PHISHING AWARENESS TRAINING

1. What is phishing?
A. A fishing technique
B. A cyberattack to steal information
C. A computer game

Enter your answer: B

Correct!

Score: 4/4
Excellent! You are highly aware of phishing threats.
```

## Learning Outcomes
- Recognize phishing emails
- Identify fake websites
- Understand social engineering tactics
- Improve cybersecurity awareness
